# Roadmap / 路线图

## 当前目标 / Current Focus

- 把项目建设为一个更可信、更容易参与的开源浏览器扩展仓库。
- 在不扩大权限范围的前提下，把当前核心下载体验打磨稳定。
- 为后续 GitHub Release、商店页整理和社区贡献打好基础。

## 近期待办 / Near-Term

- 补充更多错误提示与失败原因分类
- 增加更完整的手动验证与发布说明
- 评估权限说明与最小化策略
- 增加更多纯函数测试覆盖

## 中期方向 / Mid-Term

- 改进懒加载图片识别稳定性
- 增加结果摘要的可读性与可操作性
- 评估去重策略、失败重试和导出清单能力

## 暂不计划 / Not Planned Yet

- 站点定制爬虫逻辑
- 云同步、账号体系、远程遥测
- 与项目目标无关的复杂重构
