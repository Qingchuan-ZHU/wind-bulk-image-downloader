# Security Policy / 安全策略

## 中文说明

如果您或其他使用者发现与本项目相关的安全问题，请不要在公开 Issue 中直接披露可利用细节。

### 建议流程

1. 优先使用 GitHub 的私密安全报告功能（如果仓库已启用）。
2. 如果暂时无法使用私密报告，请通过仓库维护者的 GitHub 联系方式先行说明问题。
3. 在补丁发布前，请避免公开分享可复现的攻击细节、PoC 或受影响数据。

### 当前支持版本

| 版本       | 是否接收安全修复 |
| ---------- | ---------------- |
| `1.2.x`    | 是               |
| `<= 1.1.x` | 否，建议升级     |

## English

If you discover a security issue related to this repository, please avoid disclosing exploit details in a public issue.

### Preferred process

1. Use GitHub private security reporting if available.
2. If private reporting is unavailable, contact the repository maintainer through GitHub first.
3. Avoid publishing detailed PoCs or exploit steps before a fix is available.

### Supported versions

| Version    | Supported          |
| ---------- | ------------------ |
| `1.2.x`    | Yes                |
| `<= 1.1.x` | No, please upgrade |
