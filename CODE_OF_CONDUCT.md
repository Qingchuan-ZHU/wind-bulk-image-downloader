# Code of Conduct / 行为准则

## 中文说明

为了让项目协作保持高效、友好、可持续，请所有参与者遵守以下原则：

- 尊重不同背景、经验和观点的贡献者。
- 用问题导向、事实导向的方式讨论技术方案。
- 不进行人身攻击、羞辱、歧视、骚扰或恶意挑衅。
- 对新人保持耐心，对反馈保持具体和可执行。

如果您遇到不当行为，请通过仓库维护者的 GitHub 联系方式反馈，或在适合的情况下提交管理请求。

## English

To keep collaboration effective, friendly, and sustainable, all participants are expected to:

- Respect contributors with different backgrounds, experience levels, and opinions.
- Discuss technical decisions with a problem-solving mindset.
- Avoid harassment, personal attacks, discrimination, or hostile behavior.
- Offer actionable and respectful feedback, especially to new contributors.

If you experience unacceptable behavior, please contact the repository maintainer through GitHub.
